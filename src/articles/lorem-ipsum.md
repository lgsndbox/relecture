## Emphase et formatage du texte

Voici les différentes façons de mettre en valeur du texte :

- Texte en *italique* avec un astérisque ou _italique_ avec un underscore
- Texte en **gras** avec deux astérisques ou __gras__ avec deux underscores
- Texte en ***gras et italique*** en combinant les deux
- Texte ~~barré~~ avec deux tildes

## Listes

### Liste à puces non ordonnée

- Premier élément
- Deuxième élément
  - Sous-élément A
  - Sous-élément B
    - Sous-sous-élément 1
    - Sous-sous-élément 2
- Troisième élément

### Liste numérotée ordonnée

1. Premier point
2. Deuxième point
   1. Sous-point 2.1
   2. Sous-point 2.2
3. Troisième point

### Liste de tâches

- [x] Tâche complétée
- [x] Autre tâche terminée
- [ ] Tâche en cours
- [ ] Tâche à faire

## Citations et attributions

Pour les citations, utilisez le symbole `>`. L'attribution de l'auteur se fait sur une nouvelle ligne avec un **nom en gras** :

> La clarté est la politesse de l'homme de lettres.
>
> **Jules Renard**, *Journal*, 1906

Pour des citations plus courtes, inline :

> Le style, c'est l'homme même.
>
> **Georges-Louis Leclerc de Buffon**

Pour les citations académiques avec source complète :

> Dans chaque promenade dans la nature, on reçoit bien plus que ce que l'on cherche.
>
> **John Muir**, *Our National Parks* (1901), chapitre 1, p. 56

## Code

### Code inline

Pour mentionner une `variable` ou une `fonction()` dans le texte, utilisez des backticks.

### Bloc de code sans syntaxe

```
Ceci est un bloc de code simple
sans coloration syntaxique
  avec indentation préservée
```

### Bloc de code JavaScript

```javascript
function saluer(nom) {
  console.log(`Bonjour, ${nom}!`);
  return true;
}

const message = saluer("Monde");
```

### Bloc de code Python

```python
def calculer_somme(liste):
    """Calcule la somme d'une liste de nombres."""
    total = sum(liste)
    return total

nombres = [1, 2, 3, 4, 5]
resultat = calculer_somme(nombres)
print(f"Somme: {resultat}")
```

### Bloc de code HTML

```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Page exemple</title>
</head>
<body>
    <h1>Titre principal</h1>
    <p>Paragraphe de texte.</p>
</body>
</html>
```

## Tableaux

Les tableaux en Markdown utilisent des pipes `|` pour séparer les colonnes. La légende se place au-dessus avec *italique* ou en dessous selon la convention :

*Tableau 1 : Comparaison des langages de programmation*

| Langage    | Année | Paradigme       | Typage    |
|------------|-------|-----------------|-----------|
| JavaScript | 1995  | Multi-paradigme | Dynamique |
| Python     | 1991  | Multi-paradigme | Dynamique |
| Rust       | 2010  | Multi-paradigme | Statique  |
| Go         | 2009  | Procédural      | Statique  |

Ou avec alignement des colonnes :

*Tableau 2 : Statistiques de production*

| Année | Production | Évolution |
|------:|-----------:|----------:|
| 2023  | 1 250      | +12%      |
| 2024  | 1 400      | +12%      |
| 2025  | 1 575      | +12.5%    |

## Séparateurs horizontaux

Utilisez trois tirets, astérisques ou underscores :

---

Texte après le séparateur.

***

Autre texte après un séparateur.

## Échappement de caractères

Pour afficher des caractères spéciaux Markdown littéralement :

- Astérisque : \*
- Underscore : \_
- Hashtag : \#
- Backtick : \`
- Accolades : \{ \}

## HTML brut (si supporté)

Certains éléments HTML peuvent être directement utilisés :

<div style="padding: 1rem; background: #f0f0f0; border-left: 4px solid #666;">
  <strong>Note importante :</strong> Ce bloc utilise du HTML brut pour un style personnalisé.
</div>
